# Cornell Notes Method

Three-zone layout for maximum retention (Walter Pauk, Cornell University).

## Layout

```
┌──────────────────────┬──────────────────────────┐
│ Keywords / Questions │ Notes                    │
│ (AFTER reading)      │ (DURING reading)         │
│                      │                          │
├──────────────────────┴──────────────────────────┤
│ Summary (LAST, own words, max 5 sentences)      │
└─────────────────────────────────────────────────┘
```

## Steps

1. **Notes column** (right): Capture main ideas DURING reading
   - Capture, don't organize
   - Abbreviations, bullets, one idea per line

2. **Keywords/Questions** (left): Create AFTER reading
   - Transform each note into question or keyword
   - Active processing / retrieval
   - Use: "Que es X?", "Por que X?", "X vs Y"

3. **Summary** (bottom): Write LAST
   - 3-5 sentences, own words, no looking at notes

## Output Format

```markdown
# [Title] — Cornell Notes

| Preguntas / Keywords | Notas |
|---------------------|-------|
| Que es X? | X es... |
| Por que Y? | Porque... |
| X vs Y | Diferencia clave:... |

### Resumen
[3-5 sentences in own words]
```

## Study Tip

Cover Notes column, answer Questions from memory. Retrieval > re-reading (Karpicke 2012).
