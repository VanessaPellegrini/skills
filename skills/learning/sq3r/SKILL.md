# SQ3R Method

Robinson 1941. Active reading in 5 steps.

## Steps

### 1. Survey (S)
- Skim: TOC, headings, intro, conclusion, first sentences
- Time: 3-5 min
- Output: outline + thesis

### 2. Question (Q)
- Turn each heading into question
- Add 2-3 general questions: What about? What problem? How helps?
- Output: question list

### 3. Read (R)
- Read actively, seek answers
- Mark key passages, arguments, evidence
- Engage, don't passively highlight
- Output: key points per question

### 4. Recite (R)
- Close source. Recite answers from memory.
- Own words, no copying
- Retrieval practice step (Karpicke 2012)
- Output: recall notes

### 5. Review (R)
- Compare recitation vs source
- Identify correct vs missed
- Final synthesis
- Output: final summary + review schedule

## Output Format

```markdown
# [Title] — SQ3R Summary

## Survey
[Type, thesis, key sections]

## Questions
1. Q: [from heading] -> A: [answer]
2. Q: [from heading] -> A: [answer]
3. Q: [general] -> A: [answer]

## Recitation (from memory)
[What remembered without looking]

## Review
[What missed + final synthesis]

## Next Steps
[What to revisit, related topics]
```
