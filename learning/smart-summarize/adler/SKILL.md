# Adler's Four Levels of Reading

Progressive depth: entertain -> inform -> understand -> master.

## Level 1: Elementary (Entertain)
Basic literacy. Skip if content familiar.

## Level 2: Inspectional (Inform)
- **Systematic skimming**: TOC, headings, intro, conclusion, index
- **Superficial reading**: read quickly, skip difficult parts
- Goal: decide if deep reading worth it
- Pop-science: 90% value at this level

## Level 3: Analytical (Understand)

Four rules:
1. **Classify** — what kind of content?
2. **State thesis** — one paragraph
3. **Outline** — major parts in order
4. **Define problem** — what questions does it answer?

Additional:
5. Identify key terms + author's usage
6. Identify key propositions (main arguments)
7. Identify arguments (reasoning + evidence)
8. Did the author solve the problem?

## Level 4: Syntopical (Master)
Read multiple works on same subject:
1. Identify relevant passages across works
2. Translate terminology to own language
3. Frame own questions
4. Define disagreements between authors
5. Analyze the discussion
