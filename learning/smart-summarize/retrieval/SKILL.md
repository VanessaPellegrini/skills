# Retrieval Practice

Karpicke & Blunt 2011: retrieval > re-reading or concept mapping.

## Core Principle

Retrieving from memory strengthens the trace. Re-reading = illusion of knowledge; retrieval reveals actual knowledge.

## Steps

1. **Read once** — straight through, no notes
2. **Close source** — put away completely
3. **Free recall** — write everything remembered, any order
4. **Compare** — open source, compare recall vs original
5. **Identify gaps** — right vs. wrong vs. missing
6. **Repeat** — focus on gaps, recall again

## Key Research

- Retrieval > re-reading (Roediger & Karpicke 2006)
- Retrieval > concept mapping (Karpicke & Blunt 2011)
- Closed-book > open-book retrieval
- Feedback after retrieval critical (prevents encoding errors)
- "Desirable difficulty" — harder retrieval = stronger memory
